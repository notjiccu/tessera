# Tessera v3.0
A Rhino 8 Python script that generates a parametric rectangular grid inside any closed planar curve. Useful for layout planning, paneling, fabrication grids, and similar workflows.

---

## Features

- **Two boundary input modes** ‚Äî draw a new 3-point rectangle on the fly, or pick an existing closed curve from the scene
- **Full parametric control** ‚Äî border, columns, rows, column gap, and row gap are all set interactively
- **Cell size preview** ‚Äî shows calculated cell dimensions before committing, so you can adjust inputs if needed
- **Persistent settings** ‚Äî last used values are saved to the document and pre-filled on next run
- **Auto layer setup** ‚Äî creates a `Rectangle` parent layer (black) and `Rectangle::Cells` child layer (red) automatically
- **Optional grouping** ‚Äî all grid objects can be grouped under `GridGroup` for easy selection
- **Non-axis-aligned support** ‚Äî works correctly on rotated rectangles and curves not aligned to the world XY plane

---

## Requirements

- Rhino 8
- No external dependencies

## Running the Script

Download `Tessera_v3_0.py` and save it somewhere permanent on your machine.

**Option 1 ‚Äî RunPythonScript (no setup)**

Type in the Rhino command bar:
```
RunPythonScript
```
Browse to the file and open it. Simple, works every time.

**Option 2 ‚Äî Alias (recommended)**

Run it by typing a short name in the command bar instead of browsing every time:

1. Go to `Tools > Options > Aliases`
2. Click `New`
3. Fill in:
   - **Name:** `Tessera` (or whatever you prefer)
   - **Command macro:** `-RunPythonScript "C:\path\to\Tessera_v3_0.py"`
4. Click OK

From now on just type `Tessera` in the command bar and hit Enter.

**Option 3 ‚Äî Toolbar Button**

1. Right-click on any toolbar > `New Button`
2. Paste this into the left mouse button command field:
   ```
   -RunPythonScript "C:\path\to\Tessera_v3_0.py"
   ```
3. Add a label or icon if you want
4. Click OK

> Replace `C:\path\to\` with the actual folder where you saved the file.

## Usage

Run the script. It will walk you through the following steps:

**1. Choose boundary input method**
- `Draw3PointRectangle` ‚Äî launches Rhino's native 3-point rectangle command
- `ExistingShape` ‚Äî select any closed planar curve already in the scene

**2. Set parameters**
| Parameter | Description |
|---|---|
| Border | Inward offset from the boundary edge, equal on all sides |
| Columns | Number of columns |
| Rows | Number of rows |
| Gap X | Horizontal gap between columns |
| Gap Y | Vertical gap between rows |

**3. Confirm cell size**
The script calculates and shows the resulting cell dimensions. You can cancel here and adjust inputs if the size isn't right.

**4. Choose grouping**
Optionally group all created objects under `GridGroup`.

**5. Summary**
```
Done.
-----------------------------
Boundary:    200 x 150 mm
Border:      5 mm
-----------------------------
Columns:     4
Rows:        3
Total cells: 12
-----------------------------
Cell size:   46.250 x 46.667 mm
```

---

## Notes

- Re-running the script automatically deletes any previously created `GridGroup` before generating a new one
- The boundary curve itself is moved to the `Rectangle` layer on completion
- Settings are saved per document via `SetDocumentData`, so they persist between runs in the same file
